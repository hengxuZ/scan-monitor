### 开发背景
>大多数时候，我都会在微博上看见一些大v，晒出在链上吃赚💰的截图。比如抢盲盒、挖某矿、mint nft、抢新币等。内心总是拍断大腿，怎么我总是遇不到这些财富密码呢？于是我坚定要找到他们的私人地址，然后做好监控，等他们链上一有动态我就能第一时间发现，跟着他们一起操作。
